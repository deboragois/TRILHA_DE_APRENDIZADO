Revisão

Software de Controle de versões

Gerencia o histórico de versão local e remóto, para que tudo que está no repositório local esteja igual no repositorio remóto

TIPOS DE CONTROLES DE SOFTWARE DE CONTROLES DE VERSÃO :

CENTRALIZADO:

Possui um repositório central em um servidor com controle de versões, dependente do servidor

Distribuido:

Possui um repositório local com o controle de versões independente do servidor

GIT:

Repositório com um histórico completo e habilidade total de acompanhamento das revisões, não dependente de acesso a uma rede ou servidor central

GITHUB:

Plataforma de hospedagem de código fonte e arquivos com controle de versão
